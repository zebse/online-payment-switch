# Online Payment Switch - ZebseSwitch

A Spring Boot-based payment switch simulator built to understand the internal architecture of modern electronic payment systems.

## Goals

- Learn Spring Boot through real payment scenarios
- Simulate ISO 8583 transaction processing
- Build issuer and acquirer simulators
- Implement authorization, reversal, and settlement flows
- Explore enterprise architecture and best practices

## Technology Stack

- Java
- Spring Boot
- Maven
- PostgreSQL (later)
- Docker (later)
